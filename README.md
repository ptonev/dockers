# Dockers

#### Requirements
* Docker version 18.09.7
* docker-compose version 1.22.0

#### Installation

Go to product folder and start
```
./setup.sh
```

#### Notes

* pgAdmin4
Folder ```config``` in need to have full permissions for read/write/execute.
